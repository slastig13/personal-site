What I Have Built

Oasis

Oasis is a mobile application that challenged me and three other developers to learn and implement a new technology stack. Our application uses a Tinder-style swiping interface to maximize our users’ experience as we learn about their taste profiles. Every time a user requests a cocktail, our algorithm uses their taste profile to generate a custom recommendation. As our data set grows, our recommendations get more accurate!

Check out our presentation!

    The Stack:
  • React-Native
  • GraphQL
  • Prisma
  • Apollo
  • Node.js

smartCal

Have an hour between meetings and not sure what to do or where to go?

smartCal is a website that connects to a user's Google Calendar and provides ideas for activities to do in the area between each scheduled event on any given day. This project was completed in 3 days as a solo hack-a-thon.

    The Stack:
  • Google Calendar API
  • Foursquare API
  • Express.js
  • Passport.js
  • Redux.js
  • Node.js
  • ReactJS
